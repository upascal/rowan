# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

- `icon-72x72.png` (72x72 pixels)
- `icon-96x96.png` (96x96 pixels)
- `icon-128x128.png` (128x128 pixels)
- `icon-144x144.png` (144x144 pixels)
- `icon-152x152.png` (152x152 pixels)
- `icon-192x192.png` (192x192 pixels)
- `icon-384x384.png` (384x384 pixels)
- `icon-512x512.png` (512x512 pixels)
- `open-file.png` (96x96 pixels) - For the "Open File" shortcut
- `new-file.png` (96x96 pixels) - For the "Create New File" shortcut

## Icon Design Guidelines

The icons should follow these design guidelines:

1. Use a simple, recognizable design that represents a Kanban board
2. Use the app's color palette (#1E1E1E background, #569CD6 primary color)
3. Include a visual reference to Markdown (e.g., "MD" text or markdown symbol)
4. Ensure the design is recognizable even at small sizes
5. Use transparent backgrounds for the shortcut icons

## Creating Icons

You can create these icons using any image editing software like:
- Adobe Illustrator
- Figma
- Sketch
- GIMP
- Inkscape

After creating a master SVG or high-resolution PNG, export to all the required sizes.

## Placeholder

Until proper icons are created, you can use a simple placeholder icon with the app's name or initials on a colored background matching the app's theme.
