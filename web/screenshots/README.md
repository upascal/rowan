# PWA Screenshots

This directory should contain screenshots of the app for the Progressive Web App manifest and app stores.

## Required Screenshots

1. `screenshot1.png` - Kanban Board View (1280x720 pixels)
   - Shows the main Kanban board with columns and cards
   - Should display a realistic example with multiple columns and cards
   - Use the dark theme

2. `screenshot2.png` - File Management (1280x720 pixels)
   - Shows the home screen with recent files and action buttons
   - Should display a list of example files
   - Use the dark theme

## Additional Screenshots (Optional)

3. Card Detail View
   - Shows a card expanded with subtasks
   - Demonstrates the checkbox functionality

4. Light Theme View
   - Shows the app in light theme mode
   - Same layout as screenshot1 or screenshot2

## Screenshot Guidelines

- Resolution: 1280x720 pixels (16:9 aspect ratio)
- Format: PNG with transparency where appropriate
- Quality: High resolution, clear text
- Content: Use realistic example data, not placeholder text
- Branding: Include app name/logo in the UI
- Consistency: Use the same theme across screenshots (preferably dark theme)
